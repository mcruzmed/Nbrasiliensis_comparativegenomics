# Nbrasiliensis_comparativegenomics
Files and analysis scripts for the N. brasiliensis comparative genomics project.

We have included the following for all 26 N. brasiliensis genomes:
1. FASTA assembly files 
2. Proteome files
3. ANI matrix
4. Orthogroup statistics
5. Core and pan-genome statistics
6. Raw Newick file for phylogenetic analysis
7. Raw output files from antiSMASH (KnowClusterBlast) analysis

If you need any additional raw data files, please request them from the author.
